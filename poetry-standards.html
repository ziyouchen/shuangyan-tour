作者统一角色库

每首诗必须先生成作者角色卡，分配唯一 ID。后续所有镜头调用同一 ID，禁止重新生成人物。

角色卡必填字段

  • 姓名 / 字号
  • 年龄（诗作时期）
  • 朝代 / 身份
  • 体型 / 脸型
  • 发型 / 胡须
  • 服饰（具体颜色材质）
  • 气质 / 表情特征
  • 动作特征
  • 角色 ID（如 李白-TD-001）

角色 ID 命名规则

格式：姓名-朝代缩写-序号

  • 李白-TD-001（唐代）
  • 杜甫-TD-002（唐代）
  • 苏轼-SD-001（宋代）
  • 李清照-SD-002（宋代）
  • 王维-TD-003（唐代）
  • 白居易-TD-004（唐代）

运镜规范

每个镜头必须包含：镜头类型、运动方式、运动速度、时长。视频模型依赖这四个参数保持稳定。

推进类

  • 无人机缓慢推进（6-10秒）
  • 地面跟随推进（4-6秒）
  • 侧向横移推进（4-8秒）

拉远类

  • 垂直上升拉远（8-12秒）
  • 反向拉镜（4-6秒）
  • 广角退出（3-5秒）

环绕类

  • 人物环绕（8-12秒）
  • 山峰环绕（10-16秒）
  • 建筑盘旋（6-10秒）

固定类

  • 固定景深推移（4-8秒）
  • 呼吸感微动（3-6秒）
  • 纯静止（2-4秒）

终极继承规则

每一张图生成时自动附加以下六条继承声明。此规则解决 90% 的 AI 视频跳镜、跳风格、人物变脸问题。

01 人物继承

继承前镜头人物外形设计（角色 ID），包含脸型、体型、发型、胡须、表情特征，禁止重新生成新人物。

02 服饰继承

继承前镜头服饰设计，包含颜色、材质、配饰细节，除非诗意明确要求换装，否则全程一套服饰。

03 环境继承

继承前镜头环境 ID 及所有地貌细节，山脉走向、植被种类、地标位置不得在镜头间随机变化。

04 天气继承

继承前镜头天气设定，云量、雾浓、降水状态全部继承。若需转变则必须在提示词中明确声明"渐变"。

05 色彩继承

继承全片色板设定（六级规范），每个镜头的色彩占比不得偏离超过 15%，禁止突然高饱和或高亮度。

06 镜头语言继承

继承整体镜头语言风格（水墨动画电影），不因单句诗的意象变化而切换为写实摄影或动漫风格。

【终极继承声明 · 每个提示词末尾必须附加】 CONTINUITY DIRECTIVE: Maintain absolute visual consistency with previous shots. Same character design as 李白-TD-001 (face, body, hair, costume, accessories). Same environment design as 蜀道-ENV-001 (terrain, vegetation, atmosphere). Same weather condition (heavy fog, churning sea of clouds). Same color palette (70% ink black, 20% cool grey, 5% vermillion, 5% pale gold). Same artistic style (Chinese ink wash animation film, rice paper texture). Do NOT introduce any new visual elements not established in previous shots. This is shot N of a continuous cinematic sequence.
